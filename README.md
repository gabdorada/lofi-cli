# lofi-cli 🎧

estação de áudio minimalista para terminal. projeto prático desenvolvido para o curso de **systems analysis and development (pucpr)**.

o foco deste projeto é a automação de processos via **shell script** e o gerenciamento de fluxo de áudio em **python**, priorizando uma interface honesta e funcional.

---

### 🚀 tecnologias

- **python 3.x** (lógica e execução)
- **shell script** (automação e boot)
- **mpv / pygame** (engine de áudio)
- **git & github** (versionamento)

### 🛠️ instalação e uso
```bash
# clone o repositório
git clone https://github.com/gabdorada/lofi-cli.git

# configure o ambiente
python -m venv .venv
source .venv/bin/activate  # linux/mac
# .venv\Scripts\activate   # windows

# instale e rode
pip install -r requirements.txt
python main.py
```
